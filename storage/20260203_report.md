### 1. New explanations and inference for least angle regression
- **精简摘要**：本文提出了一种新的最小角回归（LAR）算法的解释和推断框架。通过开发新的数学性质，我们深入理解了LAR的输出行为，包括算法的适当终止点。研究发现，LAR算法在数据层面上可以视为估计一个按人口“相关性”参数递减排序的回归变量组织的响应均值的群体“路径”。此外，我们为LAR算法提供了一种新的停止规则，并通过模拟研究和数据实例展示了LAR推断方法。
- **PDF链接**：[ArXiv下载](https://arxiv.org/pdf/2602.02491v1.pdf)

### 2. Secure Multi-User Linearly-Separable Distributed Computing
- **精简摘要**：本文提出了一种新的多用户线性可分分布式计算框架，通过将计算问题转化为稀疏矩阵分解问题，实现了低计算和通信成本下的大规模并行化增益。为了保障数据安全性，采用信息论保密框架，提出了两个必要且充分的保密准则。设计了一种通用方案，适用于有限和非有限域，通过添加\(\mathrm{Null}(\mathbf{D})\)的基和注入共享随机性来保证完美信息论保密性。
- **PDF链接**：[ArXiv下载](https://arxiv.org/pdf/2602.02489v1.pdf)

### 3. The FLAMINGO Project: Exploring the X-ray--cosmic-shear cross-correlation as a probe of large-scale structure
- **精简摘要**：本研究通过FLAMINGO宇宙流体动力学模拟，探讨了宇宙剪切与弥散X射线辐射的交叉相关性，作为探测大尺度结构中气体分布的新方法。结果表明，该交叉相关性对质量在$10^{14}\leq M_{200\mathrm{c}}/\mathrm{M}_{\odot}\leq10^{15}$的晕中气体分布最为敏感，且对反馈强度敏感，但与宇宙学和重子物理变化的效应存在较大退化性。与暗能量调查的宇宙剪切和ROSAT全天X射线图的交叉相关性比较后发现，若忽略ROSAT无法解析的AGN X射线辐射，则FLAMINGO模型与数据高度一致，而强或弱反馈模型被排除；若考虑未解析的AGN，则偏好强反馈模型。因此，要充分利用X射线-引力透镜交叉相关性的潜力，需解析更暗淡的AGN，并使用外部约束来打破重子反馈与宇宙学之间的退化性。
- **PDF链接**：[ArXiv下载](https://arxiv.org/pdf/2602.02484v1.pdf)

### 4. PixelGen: Pixel Diffusion Beats Latent Diffusion with Perceptual Loss
- **精简摘要**：PixelGen是一种简单的像素扩散框架，通过感知监督优化高维像素流形，引入两种互补的感知损失，以指导扩散模型学习更有意义的感知流形。在ImageNet-256上，PixelGen在80个训练周期内实现了5.11的FID，无需VAE、潜在表示和辅助阶段，简化了生成范式。代码公开于https://github.com/Zehong-Ma/PixelGen。
- **PDF链接**：[ArXiv下载](https://arxiv.org/pdf/2602.02493v1.pdf)

### 5. RLAnything: Forge Environment, Policy, and Reward Model in Completely Dynamic RL System
- **精简摘要**：我们提出了RLAnything，一个通过闭环优化动态塑造环境、策略和奖励模型的强化学习框架，增强了任何大型语言模型或智能体场景下的学习信号和整体RL系统。策略通过步骤和结果信号的集成反馈进行训练，奖励模型通过一致性反馈联合优化，进而提升策略训练效果。理论驱动的自动环境适应利用每个模型的批评反馈进行训练，实现经验学习。实证结果显示，每个新增组件都持续提升了整体系统性能，在OSWorld上Qwen3-VL-8B-Thinking提升了9.1%，在AlfWorld和LiveBench上Qwen2.5-7B-Instruct分别提升了18.7%和11.9%。优化的奖励模型信号优于依赖人工标签的结果。代码：https://github.com/Gen-Verse/Open-AgentRL。
- **PDF链接**：[ArXiv下载](https://arxiv.org/pdf/2602.02488v1.pdf)

### 6. Deep Learning-Based Object Detection for Autonomous Vehicles: A Comparative Study of One-Stage and Two-Stage Detectors on Basic Traffic Objects
- **精简摘要**：本研究探讨了基于深度学习的自动驾驶车辆对象检测技术，比较了一阶段检测器YOLOv5和两阶段检测器Faster R-CNN在基本交通对象识别上的性能。通过实验分析，发现YOLOv5在mAP、召回率和训练效率上表现更优，尤其在数据集规模和图像分辨率增加时。而Faster R-CNN在检测小而远的物体以及在挑战性光照条件下表现更好。研究为自动驾驶系统选择合适检测方法提供了指导。
- **PDF链接**：[ArXiv下载](https://arxiv.org/pdf/2602.00385v1.pdf)

### 7. Emergent Cooperation in Quantum Multi-Agent Reinforcement Learning Using Communication
- **精简摘要**：本文将通信方法应用于量子多智能体强化学习中的量子Q学习智能体，包括相互认可令牌交换（MATE）协议及其扩展MEDIATE、同伴奖励机制Gifting和强化智能体间学习（RIAL）。在三个顺序社会困境中评估这些方法，结果显示使用MATE和时间差分度量（MATE\textsubscript{TD}）、AutoMATE、MEDIATE-I和MEDIATE-S的方法在所有困境中均实现了高水平的合作，证明了通信是促进量子多智能体强化学习中涌现合作的有效机制。
- **PDF链接**：[ArXiv下载](https://arxiv.org/pdf/2601.18419v1.pdf)

### 8. Conduit: Programmer-Transparent Near-Data Processing Using Multiple Compute-Capable Resources in Solid State Drives
- **精简摘要**：本文提出了一种名为Conduit的通用、透明的近数据处理（NDP）框架，适用于固态驱动器（SSD）。Conduit利用SSD内的多种计算资源，通过编译时向量化和运行时指令卸载决策，提高计算效率。在六个数据密集型工作负载上，Conduit比现有最佳卸载策略性能提升1.8倍，能耗降低46%。
- **PDF链接**：[ArXiv下载](https://arxiv.org/pdf/2601.17633v1.pdf)
